# Handy Tools

A growing collection of small, focused browser utilities deployed on **Cloudflare Pages**.  
No accounts. No tracking. Everything runs client-side where possible.

🌐 **Live site:** [myexample.work](https://myexample.work)

---

## Apps

### 💬 LINE Chat Converter
> `line-export-converter/`

Ever tried reading a raw LINE `.txt` export? It's a wall of text. This tool transforms it into a beautiful, WhatsApp-style HTML chat log you can actually read and share — right in your browser, with zero upload.

Export from LINE → drop the file → get a stunning `.html` in seconds.

- **🔒 100% private** — your chat data never leaves your device, ever
- **🎨 Styled chat bubbles** — colour-coded per sender with initials avatars, your messages on the right
- **📅 Date & time separators** — easy to skim through long conversations
- **🖼️ Media placeholders** — stickers, photos, videos and files shown as labelled badges instead of broken links
- **🌏 Bilingual** — fully supports Traditional Chinese (HK/TW) and English LINE exports
- **📦 Self-contained download** — one `.html` file you can open anywhere, forever, no internet needed

### 💬 WhatsApp Chat Converter
> `whatsapp-export-converter/`

Your WhatsApp memories deserve better than a `.txt` file. Drop your export and get a clean, familiar-looking chat log — the kind you'd actually want to read or keep.

Export without media from WhatsApp → drop the file → download your `.html`.

- **🔒 100% private** — nothing is uploaded, all processing happens in your browser
- **🎨 Familiar chat UI** — green bubbles, sender names, timestamps, just like the real app
- **👥 Multi-participant** — colour-coded avatars for every member in group chats
- **🌏 Bilingual** — supports Traditional Chinese (HK/TW) and English exports
- **📦 Self-contained download** — share or archive as a single portable `.html` file

## Dashboards

### 🍣 Sushiro Queue Status
> `sushiro-hk/`

Tired of opening the Sushiro app just to see if it's worth the trip? This dashboard shows live queue waiting times across every Sushiro HK branch on one page — at a glance, no login, no app.

- **⚡ Real-time data** — live queue figures pulled directly, auto-refreshed so you're never looking at stale numbers
- **🗺️ All branches at once** — every HK location on a single screen, no tapping branch by branch
- **📊 At-a-glance comparison** — instantly spot which branch near you has the shortest wait
- **🈶 Traditional Chinese interface** — built for HK locals
