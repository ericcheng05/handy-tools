## Apps

### 💬 LINE Chat Converter
> `line-converter/`

Converts a LINE group chat `.txt` export into a polished, readable HTML file — styled with avatars, colour-coded senders, date separators, media placeholders, and dark-mode design.

**How to use:**
1. Open LINE on your phone → Settings → Chats → Export chat history → Save as `.txt`
2. Go to [tools.hlcheng.cf/line-converter](https://tools.hlcheng.cf/line-converter)
3. Drop the `.txt` file, enter your display name, click **Convert**
4. Preview the result, then download the `.html` file

**Features:**
- Fully client-side — your chat data never leaves your browser
- Supports Traditional Chinese (HK/TW) and English LINE exports
- Right-side bubbles for your own messages
- Colour-coded avatars per member
- Sticker / photo / video / file placeholders
- Download a self-contained `.html` + `.css` bundle
